## Description
Se tiene implementar un componente de caja de texto (textbox) que permita a los usuarios ingresar y editar texto. El componente debe ser accesible, adaptable a diferentes tamaños de pantalla y compatible con temas visuales. Debe soportar características como selección de texto, cursor parpadeante, y eventos de entrada.

## Acceptance criteria
- [ ] El textbox debe poder aceptar cualquier tipo de elemento a su ziquierda. Botones iconos y demas. Y debe adaptarse a ellos sin problemas. Esto es importante para que el textbox pueda ser usado en diferentes contextos y con diferentes tipos de contenido.
- [ ] El textbox debe poder aceptar cualquier tipo de elemento a su derecha. Botones iconos y demas. Y debe adaptarse a ellos sin problemas. Esto es importante para que el textbox pueda ser usado en diferentes contextos y con diferentes tipos de contenido.
- [ ] El textbox debe ser frienly con web frameworks como React,Vue, Svelte, Angular, Solid.
- [ ] Se debe hacer test extensivos sobre el componente.
- [ ] Se debe tener un demo donde se integre con otros componentes como UIWindowWC, UIButtonWC, etc. para mostrar su versatilidad y adaptabilidad en diferentes contextos.
- [ ] Debe tener varrios estilos. Visuales, label flotante, label fija, Anexo imagenes. Y https://flowbite-svelte.com/docs/forms/floating-label
